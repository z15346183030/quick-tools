# 极简工具箱 - 免费在线工具站

## 包含工具
- JSON 格式化 / 压缩 / 校验
- 文本对比（逐行 diff）
- Base64 编解码
- 时间戳转换
- 正则表达式实时测试
- Markdown 实时预览
- 颜色转换（HEX/RGB/HSL）
- 字数统计

## 特点
- 纯前端，零后端，免费部署
- 暗色主题，现代 UI
- 移动端适配
- SEO 友好
- 数据不上传，隐私安全

## 部署方式（任选一个，全部免费）

### 1. Vercel（推荐）
1. 注册 https://vercel.com
2. 把这个文件夹拖进去
3. 自动部署，得到一个 `.vercel.app` 域名

### 2. Netlify
1. 注册 https://netlify.com
2. 拖拽文件夹部署
3. 得到 `.netlify.app` 域名

### 3. GitHub Pages
1. 上传到 GitHub 仓库
2. Settings → Pages → 选择 main 分支
3. 得到 `username.github.io/repo` 域名
